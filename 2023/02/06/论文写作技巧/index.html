论文写作技巧

无标签
发布日期:   2023-02-06

一、学术论文两要素：专业的知识、科学的表达

1. 专业的知识包括以下特点（你写的这篇文章要具备的要素）

  • 新颖性：创新点

  • 普适性：提出的方法、理论、发现、实验等不仅适用于单一的情况

  • 学术性：从项目中提炼科学问题

2. 科学的表达

  • 清晰的表达思路：想好文章的中心思想，为了论证它怎么摆事实、讲道理；（围绕基于xx的一种方法，围绕方法进行验证，实验和结果展示（摆事实），这些结果说明了这个方法好（讲道理））
  • 良好的文章结构：题目、摘要、前言、问题描述、方法、验证、结论
  • 公认的科学语言：要用业内的专业术语来表述提出的方法或理论

二、论文选题

1. 阅读国内外本领域高水平的大学博士论文

可以获得第一手全面的背景、方法、逻辑以及进展

2. 广泛阅读高水平期刊论文（学会检索文献）

看什么决定想什么，做什么，如何做

3. 确定论文的方向，确定几个可能的内容

  • 很难在选题阶段就将要做的内容“算计”很全面
  • 关键是把握方向，研究过程中不停地修正
  • 很多论文与选题报告相差甚远

三、选题建议

1. 本领域国际学术界研究热点但未能突破的问题
模仿别人，跟风随大流，容易出成果，比较难出高水平成果（跟别人解决一样的问题，使用不同的方法，并且解决得比他好）

2. 基于课题组的学术成果进行拓展延伸（做的好的论文可以进行拓展研究）
可能并非当前热门的领域，但研究扎实，论文水平高
3. 当前工程中尚未解决的难题（项目过程中发现的问题）
需要提炼科学问题，比较难，很有价值
4. 考虑课题组的实验条件、经费及个人兴趣

四、如何研究

  1. 尽快建立自己的信心和兴趣：建立正反馈机制，不断制定小目标然后实现，逐渐找到科研的感觉；
  2. 要制定严格的计划：计划完不成可以修正，但没计划可能无限拖延；
  3. 不放过任何灵感：时刻记录下自己的想法，不成熟的可以完善，不可行的再放弃；
  4. 研究的内容可以并行：不要被一个点卡住就止步，先做别的，再回过头来想新的方法。

五、创新点

干了别人没干过的事，就是创新

  1. 发现别人没发现的现象
  2. 提出别人没提出的理论或方法
  3. 解决了别人没解决的问题
  4. 设计了别人没做过的实验

六、创新的境界

  1. 无中生有（最高境界）
  2. 标新立异：原始创新，发现新问题，提出新方法；
    **3. 新瓶装旧酒，旧瓶装新酒：提出新方法解决旧问题，发现新问题使用旧方法解决。
  3. 移花接木：旧问题，旧方法，新的排列组合**
  4. 批判式创新
    指出别人文献中的问题，并对其作出了解释。

七、基本学术道德和规范必须遵守

八、文献检索

  • 中文检索库：
  • 中国知网、东北大学学位论文（https://202.118.8.24/）
  • 外文检索库：IEEE(IEEE Xplore)、Elsevier(www.sciencedi)、Springer(https://link.springer.com/)、SAGE(https://journals.sagepub.com/)、
  • SCI-Hub论文下载：(https://tool.yovisun.com/)
  • 检索收录：
    EI：https://www.engineeringvillage.com/
    SCI：web of science: https://www.webofscience.com/
    SCI分区：http://www.letpub.com.cn/index.php?page=journalapp

九、阅读文献的目的

  1. 研究该领域的背景和进展
  2. 了解研究热点和前沿
  3. 寻找前人研究的不足、缺陷
  4. 学习论文的写作技巧
  5. 学习他人提出的优秀的方法

步骤：

  1. 找5～10篇中文的博士论文和综述型期刊论文
    博士论文：主要了解领域的背景、进展、主要研究方法和手段等，3～7天一篇，英文论文1-2周一篇，重点关注前言和用了哪些方法，具体方法的细节看不懂没关系
  2. 研究型论文（顶刊顶会）
    了解领域内的热点、前沿，学习先进新颖的想法、方法，学习写作技巧，1周1篇到1天-N篇，复现1-3篇经典高质量论文

十、研究型论文阅读方法

  1. 看题目，研究对象和方法（1）
  2. 看作者，有没有大牛（10s）
  3. 看摘要，具体是什么问题，用了什么方法，取得了什么效果(5~10)
  4. 前言，建模，方法，验证统统过
  5. 看原理图，整体框架图（3～5）
  6. 方法和我们的方向关系密切的话就仔细看，公式推导和细节（20～30）
  7. 看图，是否表达了需要表达的信息（画得好的可以仿照着画，2～3）
  8. 看结论，贡献和效果

十一、引用论文的技巧

  1. 数量要多（全面），20起步，30合格，40优秀
  2. 质量要高（水准）：
    高水平SCI期刊论文：参考文献的主体构成
    • 权威会议的论文：不要过多引用，0～5篇
    • 经典、权威的英文专著：适当引用1～2部
    • 中文文献和非SCI论文：能少则少

十二、写作建议

  1. 一气呵成
  2. 写作和实践相结合：白天写论文，晚上做实验/调仿真
  3. 制定计划，严格执行，尽快完成初稿
  4. 反复修改（修改10遍）

十三、写作顺序

  1. 题目/摘要（初步写一个）
  2. 问题描述/建立模型
  3. 方法和验证分析
  4. 结论
  5. 前言
  6. 摘要（认真修改）：表述前因后果讲清楚、创新点更加详细、贡献更加突出
    • 描述对象/问题及其重要性
    • 说明难点或者未解决的难题
    • 本论文提出了什么方法
    • 提出的方法是怎样实现的/解决问题的
    • 方法取得了什么效果，有什么贡献，有哪些拓展应用，未来还能怎么研究等等
  7. 题目（反复推敲）

十四、新手第一篇——先抄袭再超越

  1. 找一篇与自己研究内容最类似的高水平论文
  2. 反复阅读（3～5遍）
  3. 学习论文中的表达方式（问题描述、结果分析等）
  4. 对照着写论文（卡壳了就看看别人怎么写的）
  5. 过程中要想清楚自己的论文的优势
  6. 写完反复修改语言
  • 题目：
  1. 对象+问题+方法
    XXX（Method）of XXX (object) for XXX(problem)
  2. 对象/问题（可以简化）+方法（描述其创新性、独特性）用这个
    XXX（Method）of XXX（object/problem）with/based on/considering XXX(characteristics)
  3. 对象/问题+方法（描述其创新性和普适性）
    XXX(Method) of XXX（object/problem）with/based on/considering XXX(characteristics) + ??(设计和实验)

十五、前言

  1. 交代研究背景
  2. 说明研究内容的重要性
  3. 目前研究的不足
  4. 提出研究的问题
  5. 简要说明解决方法
  6. 说明为什么用上述方法
  7. 创新点和贡献
  8. 给出文章结构

内容：

  1. 问题背景
  2. 研究进展
  3. 问题描述
  4. 动机和创新点
  5. 文章结构（有加有不加）

十六、介绍他人工作

吹：正面积极地评价他人工作，使用修饰词，如new、novel、firstly、greatly、effective、important等，篇幅长
黑：委婉前人研究的不足，引出本文的工作（研究的问题），少用but,用however，不要直接评论他人工作不足，而是说某个问题前面研究没考虑。话不要说死，留后路，as far as authors/we know

十七、建模和问题描述

归纳建立一个简洁又不失深度的问题/数学模型
1. 建立模型
自己提出的模型，要重点描述
引用别人的就简单描述
自己的改进的部分，重点描述改进的部分
2. 归纳问题
科学问题，有学术性，专业的学术语言

十八、仿真和实验

设计一个有说服力的实验/仿真验证，选择具有代表性的，可以出好看的图片的实验
多进行消融实验，结果分析充分详尽（可以学习别人的）

十九、结论

总结一个有学术价值的结论
本文针对XX问题，提出了XX方法，从而解决了其中XX难题，取得了XX效果（有数据最好）。
本文的主要贡献在于：1）提出了XX方法，和之前研究相比有什么特点或创新；2）解决了什么问题，克服了哪些困难，取得了什么样的效果3）该方法还能扩展应用到哪些方面。
本文工作还有哪些不足，未来工作将从哪几个方面进行。（要委婉）
