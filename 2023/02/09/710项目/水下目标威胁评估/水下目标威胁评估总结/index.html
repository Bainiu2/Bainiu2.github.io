# 一、主要任务

根据状态估计获得的目标的位置和运动状态信息，对水下运动目标（移动声靶、蛙人）进行威胁评估，给出目标对 AUV 的威胁程度。

# 二、威胁评估指标选择

距离（AUV 与目标）、碰撞风险、航向差、速度差、AUV 数量（同类目标数量）

# 三、主要方法
